# Summary

This mount for smart home tablet for the wall. Thickness mounting only 2mm.

The first part is mounted on the wall with self-tapping screws.
The second part is attached to the tablet on the double-sided tape.

The second part insert into the first, through the mini rail system, which holds the tablet and allows it to be easily removed from the wall.
