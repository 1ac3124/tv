# Summary

This model is created to calibrate junction deviation, linear advance or jerk settings. You can either print it as it's provided with different values, or stretch it over Z axis and dynamicly change values over height.

----------------

Эта модель создана чтобы калибровать junction deviation, linear advance и jerk. Вы можете как распечатывать эти модели как есть, придавая им разные настройки, либо растянуть вдоль оси Z и динамически менять параметры.
