# Summary

###Simple and strong spool holders for 2020 aluminium profiles

###Advantages:

- Can be printed without bridges and supports
- Strong enough to sustain even a 2.5KG spool
- Needs only 1 screw and nut
- Can be installed on any place on 2020 aluminium V-slot profile
- Can be installed on any other profiles that uses slots from 2020 v-slot profile
- Can be installed not only vertically, but also horizontally

Things needed for assembly:
- 1 M4x10 screw
- 1 M4 slot nut

# Print Settings

Printer: Tevo Tornado
