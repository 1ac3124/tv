# Summary

## Z motor mount for Creality3D Ender-3 or any similar 3D printer

Created to be easy in print and not to block motor cooling. Longer version is for installing motor with two 3mm dampers.

### List of parts for assembly:
* 3x M4x8 
* 3x M4 slot nut
