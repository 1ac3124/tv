# Summary

###40mm fan cooling modules for K3D modular cooling system

This is modules for use with K3D modular cooling system. If you want to use it on your printer, you will also need the base module. Curently there is only 2 base modules:

- for Tevo Tornado : https://www.thingiverse.com/thing:2808673
- for e3d chimera upgrade for Tevo Tornado/Creality CR-10: https://www.thingiverse.com/thing:2937975

But you can create base module for your 3d printer and use all K3D cooling system modules!

For assembling you will also need:

- 40x40x10 fan

# Print Settings

Printer: Tevo Tornado
Rafts: Doesn't Matter
Supports: No
Resolution: 0.4 or better
Infill: any

Notes: 
Print upside-down with bridging enabled. Preferably use ABS, PETG or other heat-resistant plastics. If you print from the PLA - it will work, but in time it can lose its shape due to high temperatures.

###Warning!
Use horizontal size compensation setting of your slicer. The recommended value is -0,5mm, can be changed to achieve a tight fit of the modules. If you print with a value of 0mm, then the modules most likely will not be connectable.
