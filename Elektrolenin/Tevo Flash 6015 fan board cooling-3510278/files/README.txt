# Summary

###Tevo Flash 6015 fan board cooling

This modification is created to make a reliable cooling system for board. One 60x60x15 fan i used to blow cold air on board and stepper motor drivers, other to remove hot ait from electronics box. 

To install it you will need to cut case cover as it is drawn on photo.

============

Эта модификация создана чтобы обеспечить надёжное охлаждение платы принтера. Один 60х60х15 вентилятор дует на плату и драйверы, второй выводит горячий воздух из корпуса.

Для установки вам надо будет вырезать отверстия в крышке как показано на фото.
