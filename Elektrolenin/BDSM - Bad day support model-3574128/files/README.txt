# Summary

###BDSM - bad day support model

This model is created to test different aspects of support parameters. It has 3 tests:
* Small area support test
* Detach test - big area with narrow mount
* Enshroud test - small object between 2 big.

----------------

Эта модель создана, чтобы протестировать различные аспекты построения поддержек. В ней есть 3 теста:
* Тест поддержек для элементов с малой площадью
* Тест на отеделение - большая пластина на тонкой ножке
* Тест на окутывание - небольшой объект между двух больших.
