# Summary

#UNI extruder - K3D version

It's a remix of UNI extruder. Optimized for easier printing and increased strength. Also, direct version included

##Parts for assembly:
* mk8 or ultimaker extruder gear
* 3x M3 screws. Length differs from type and mount.
* Spring. ~1mm thick, 8-12mm diameter, 10-15mm length.
* 1x 623zz bearing
* 2x f623zz or 623zz bearings
