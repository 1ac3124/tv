# Summary

###BLTouch sensor mount module for K3D cooling system

This is modules for use with K3D modular cooling system. If you want to use it on your printer, you will also need the base module. Curently there is only 2 base modules:

- for Tevo Tornado : https://www.thingiverse.com/thing:2808673
- for e3d chimera upgrade for Tevo Tornado/Creality CR-10: https://www.thingiverse.com/thing:2937975

But you can create base module for your 3d printer and use all K3D cooling system modules!

Different versions of this module is created to make it easier to calibrate z offset of BLTouch sensor. Basicaly you will need Z0 version, but if your Z offset will be more than 0 or less than -1 then you will need to compensate it via other versions. For example you want to move BLTouch sensor down for 2 mm. Than you will need Z-2 version of module.

For assembling you will need:
- BLTouch or 3DTouch sensor
- 2x M3x10 or longer screws
- 2x M3 nut
