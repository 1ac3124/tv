# Summary

This is cooling system for e3d v6 hotend and nozzle. All fans and screws are the same except for BLTouch mount

Original Fusion 360 files here: https://a360.co/2JB2IfC

Update v4:
- Changed base part front panel perforation to make it more rigid
- Optimized fan duct for printing

Version 4 is fully tested on my Tevo Flash. BLTouch Z offset is -0.7mm. With e3d v6 hotend printer works much better, than with volcano.

###Only zone parts is changed to e3d v6. Cold parts (heat sink and throat) remains stock.

-------

Это система охлаждения для e3d v6 хотэнда. Все кулеры и крепёж остались неизменны, если не считать крепёж BLTouch.

Оригинальные файлы Fusion 360: https://a360.co/2JB2IfC

Update v4:
- Изменены прорези на передней части базовой детали, чтобы сделать её более жёсткой
- Оптимизировано сопло обдува модели, чтобы было легче печатать.

Версия 4 полностью протестирована на моём принтере. Z offset для BLTouch откалибровался около -0.7мм. С e3d v6 хотэндом печатать стало значительно проще, качество моделей возросло.

###Меняется на е3д в6 только нагревательный блок и сопло. Радиатор и горло остаются стоковые.
