# Summary

12864 LCD adjustable mount for 2040 aluminium profile
===========

This mount was created to allow the display to be installed anywhere on the 2040 frame. It is adjustable for tilting and turning.

###For assembly you will need:
- 4x M3x12 screw
- 2-4x M4x8 screw and nut to attach mount to 2040 aluminium v-slot frame
