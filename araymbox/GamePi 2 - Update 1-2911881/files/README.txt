# Summary

This is version 2.1 of my GamePi handheld console.

If you like the GamePi check out my page https://www.fb.com/araymBox for other versions and the upcoming devices. You can also post ideas on how to improve the design.

--------------------------------------------- 
It's a mix of all my GamePi Versions and combines the best features:
- GamePi 1 ( https://www.thingiverse.com/thing:2758962 )
- GamePi 2 ( https://www.thingiverse.com/thing:2809128 )
- GamePi Zero ( https://www.thingiverse.com/thing:2823313 )

If you want to rebuild this GamePi check out the instrucatables of the GamePi 2 and GamePi Zero. I won't write another for this one. 

https://www.instructables.com/id/GamePi-the-Handheld-Emulator-Console/
https://www.instructables.com/id/GamePi-Zero-the-Favorable-Emulation-Station/


--------------------------------------------- 
To avoid any further problems with powerbanks not being available anymore this case is designed for the separate powerbank-electronics and batteries: 
- 1x PCB ( https://www.aliexpress.com/item/5V-2-1A-Power-Bank-Charger-Module-Charging-Circuit-Board-PCB-Step-Up-Boost-Power-Module/32792636437.html )
- 2x 18650 LiPos ( https://www.aliexpress.com/item/4PCS-liitokala-For-samsung-18650-3500mAh-13A-discharge-INR18650-35E-INR18650-35E-18650-battery-Li-ion/32807017443.html ) 

--------------------------------------------- 
The buttons Start and Select do not need to be printed anymore since they take too much space. use 6x6mm tactile switches with a 6mm long button, e.g.:
https://www.aliexpress.com/item/50pcs-6-6x4-3mm-6X6X5mm-DIP-Tactile-Tact-Mini-Push-Button-Switch-Micro-Switch-Momentary-DIP/32842271522.html

--------------------------------------------- 
You can leave a tip (button left) if you like my stuff.
