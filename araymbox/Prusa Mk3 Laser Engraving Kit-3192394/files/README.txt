# Summary

These are remix parts (and some extras) of marzsolts laser engraver mount featured in this instructable: 

https://www.instructables.com/id/Laser-Engraving-on-Your-Prusa-MK3/

I remixed the mount so the laser sits on the left side of the extruder motor. This had to be done because of Prusas new 45 degree fan shroud design. 

This is the laser I used: 
https://bit.ly/2JzGGGh

The power supply is 12V 2A I had laying around.
