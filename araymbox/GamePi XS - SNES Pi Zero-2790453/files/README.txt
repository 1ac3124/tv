# Summary

This is a back cover for a fake SNES controller. It can be used to create a raspberry pi zero based portable game station.
It can be used with any HDMI-TV/Display.

If you want to rebuild this GamePi check out my instructable:
https://www.instructables.com/id/GamPi-XS-the-PlugnPlay-Emulation-Station
