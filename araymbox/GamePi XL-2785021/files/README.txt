# Summary

Check out the new version with joysticks:
https://www.thingiverse.com/thing:2809128

This is a case to build a raspberry pi based portable game station.
It was designed around the offical 7" Display (https://goo.gl/xrHUcS) and a Raspberry Pi 3.

The case is split into 11 parts so it can be printed on smaller printers. The parts can be glued together. 

Updates:
- 13.02.2018: Added the archive "GamePi_XL_not_split.zip" which contains the not split STLs
- 15.02.2018: Added "GampePi_XL_controls_R_2nd_stick.stl" for a device with 2 analog sticks



For the build shown in the pictures I've use the following parts:

- 7" Display (https://goo.gl/MibvNa)
- Raspberry Pi 3 (https://goo.gl/Ype2vc)
- microSD Card 16GB (https://goo.gl/Tj5qQE)
- Power (Electronics/Batteries) (https://goo.gl/NTsSNZ)
- Controller ((Electronics/Buttons) (https://goo.gl/nYanBP)
- Stereo Amp 2x1,5W (https://goo.gl/3UiiyA)
- Soft Tactile Switches (https://goo.gl/nMuzaF)
- 3,5mm switched Stereo Jack
- small 1,5W Speakers from an old notebook
- other small parts and cables
