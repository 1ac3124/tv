# Summary

This is a low cost (LC) version of my GamePi 2 ( https://www.thingiverse.com/thing:2809128 ) handheld console.

If you like the GamePi check out my page https://www.fb.com/araymBox for other versions and the upcoming devices. You can also post ideas on how to improve the design.

It's a simple sleek shell to create a raspberry pi based portable game station.
It was designed around a 4.3" car rear view display and a Raspberry Pi Zero W.

If you want to rebuild this GamePi check out my instructable:
https://www.instructables.com/id/GamePi-Zero-the-Favorable-Emulation-Station

You can leave a tip (button left) if you like my stuff.
