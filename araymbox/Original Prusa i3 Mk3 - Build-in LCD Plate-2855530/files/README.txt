# Summary

It's a remix of the original Prusa i3 Mk3 LCD cover (revision R2): 

https://github.com/prusa3d/Original-Prusa-i3/blob/MK3/Printed-Parts/stl/LCD-cover-ORIGINAL-MK3.stl

It's designed to be build into a printer enclosure and adds holes for two switches (power and light).

Cutout LCD cover: 194mm x 70mm
Cutout power switch: 31mm x 23mm
Cutout light switch: 20mm x 13mm
