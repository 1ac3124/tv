# Summary

Check out the new version with joysticks:
https://www.thingiverse.com/thing:2809128

This a simple sleek shell to create a raspberry pi based portable game station. 
It was designed around a 5" Elecrow Display and a Raspberry Pi 3. 

I've added two versions of the shell:
1st: Front and back cover are made from one part (fits fine on a 21x21cm bed)
2nd: Front and back cover are split into 3 parts each (for smaller printers)

For the build shown in the pictures I've use the following parts:
- 5" Display (https://www.amazon.de/gp/product/B013JECYF2)
- Raspberry Pi 3 (https://www.amazon.de/dp/B01CD5VC92)
- microSD Card 64GB (https://www.amazon.de/gp/product/B073SB2L3C)
- Power (Electronics/Batteries) (https://www.amazon.de/gp/product/B00XTVOAQM)
- Stereo Amp 2x1,5W (https://www.ebay.de/itm/271513588858)
- Soft Tactile Switches (https://www.amazon.de/gp/product/B01JD3H5RK)
- Buttons from an old Controller
- 3,5mm switched Stereo Jack 
- small 1,5W Speakers from an old notebook
- other small parts and cables

You can leave a tip (button left) if you like my stuff.
